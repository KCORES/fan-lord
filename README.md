Fan Lord
--------

[README in English](README_en-US.md)

一个用于控制 Supermicro X 系列主板风扇的图形界面工具。

本项目是一个 [KCORES 开源项目](https://github.com/KCORES)。

## 功能特性

- 控制 Supermicro X 系列服务器主板的风扇
- 显示风扇状态

## 目标

- 为所有具有 IPMI 接口的主板提供简单易用的风扇控制图形界面
- 欢迎为本项目做出贡献
